### API

- Employee
- Create
    - POST /api/employees
        - Create a new employee
- READ
    - GET /api/employees
        - Read a list of employees
    - GET /api/employee/ {employeeId}
        - Read a single employee
- UPDATE
    - PUT /api/employees
        - Update an existing employee
- DELETE
    - DELETE /api/employees/{emplyeeId}
        - Delete an existing employee
